# 图与景区信息管理系统

基于图算法实现的景区信息管理系统，提供景点导航、路径规划、电路铺设等功能。

## 实验目的

通过"景区信息管理系统"的编程实践，学习图的遍历、Dijkstra算法、最小生成树算法、Prim算法和它们的编程应用。

- 掌握图的定义和图的存储结构
- 掌握图的创建方法
- 掌握图的两种遍历方法
- 理解迪杰斯特拉（Dijkstra）算法
- 理解最小生成树的概念和普里姆（Prim）算法
- 掌握文件操作
- 使用C++语言，利用图的数据结构，开发景区信息管理系统

## 业务背景

现有一个景区，景区里面有若干个景点，其景点分布如图1所示，并且景点之间满足以下条件：

1. 某些景点之间铺设了道路（相邻）
2. 这些道路都可以双向行驶的（无向图）
3. 从任意一个景点出发都可以游览整个景区（连通图）

开发景区信息管理系统，对景区的信息进行管理。使用图的数据结构来保存景区景点信息，为用户提供创建图、查询景点信息、旅游景点导航、搜索最短路径、铺设电路规划等功能。

## 景点数据

景区的数据包含景点信息和景点之间的道路信息，分别由两个文本文件存储。

### 景点信息（Vex.txt）

景点信息主要包含景点的编号、景点的名字和相关介绍等内容：

| 编号 | 名字 | 介绍 |
|------|------|------|
| 0 | A区 | ... |
| 1 | B区 | ... |
| 2 | C区 | ... |
| 3 | D区 | ... |
| 4 | E区 | ... |
| 5 | F区 | ... |
| 6 | G区 | ... |

### 道路信息（Edge.txt）

道路信息通过一个三元组的形式来表示，分别是景点1（起点）的名字，景点2（终点）的名字，两个景点之间的距离长度：

| 景点1 | 景点2 | 距离（m） |
|-------|-------|-----------|
| A | C | 700 |
| A | E | 1000 |
| A | F | 600 |
| B | C | 1000 |
| B | G | 1000 |
| C | D | 400 |
| D | E | 300 |
| D | G | 400 |
| E | F | 600 |
| F | G | 500 |

## 系统功能

整个景区信息管理包含5个功能模块，分别是创建图，查询景点，旅游景点导航，搜索最短路径和铺设电路规划。

### 1. 创建图

**输入：** 从Vex.txt文件中读取景点信息，从Edge.txt文件中读取道路信息。

**处理：** 根据读取的景点信息创建景区景点图。

**输出：** 创建成功，一次输出：
1. 顶点数目
2. 顶点编号
3. 顶点名字
4. 边两端的顶点编号
5. 边的权值

创建失败，输出失败的提示信息。

### 2. 查询景点

**输入：** 想要查询的景点的编号

**处理：** 根据输入的景点编号，查询该景点及相邻景点的信息。

**输出：**
1. 景点名字
2. 景点介绍
3. 相邻景点的名字
4. 到达相邻景区的路径长度

### 3. 旅游景点导航

**输入：** 起始景点的编号

**处理：** 使用深度优先搜索（DFS）算法，查询以该景点为起点，无回路游览整个景区的路线。

**输出：** 所有符合要求的导航路线。

### 4. 搜索最短路径

**输入：** 起始景点的编号和终点的编号

**处理：** 使用迪杰斯特拉算法，求得从起始景点到终点之间的最短路径，计算路径总长度。

**输出：**
1. 最短路线
2. 路径总长度

### 5. 铺设电路规划

**输入：** 景区的所有景点信息和道路信息

**处理：** 根据景区景点图使用普里姆（Prim）算法构造最小生成树，设计出一套铺设线路最短，但能满足每个景点都能通电的方案。

**输出：**
1. 需要铺设电路的道路
2. 每条道路铺设电路的长度
3. 铺设电路的总长度

## 核心算法

### 深度优先搜索（DFS）
用于旅游景点导航，查找无回路的游览路线。

### 迪杰斯特拉（Dijkstra）算法
用于搜索最短路径，计算两点间的最短路径和距离。

### 普里姆（Prim）算法
用于铺设电路规划，构造最小生成树，找到总长度最小的电路铺设方案。

## 数据结构

### 顶点（Vex）
```cpp
typedef struct Vex {
    int num;           // 景点编号
    std::string name;  // 景点名称
    std::string desc;  // 景点描述
} Vex;
```

### 边（Edge）
```cpp
typedef struct Edge {
    int vex1;   // 顶点1编号
    int vex2;   // 顶点2编号
    int weight; // 边权重
} Edge;
```

### 邻接表节点（ArcNode）
```cpp
typedef struct ArcNode {
    int adjvex;  // 邻接顶点下标
    int weight;  // 边权重
} ArcNode;
```
